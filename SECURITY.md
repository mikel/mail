# Security Policy

This project is maintained by [Ryan Bigg](https://ryanbigg.com).

## Supported Versions

| Version | Security Patches |
| ------- | ---------------- |
| 2.9.x   | Yes              |
| 2.8.x   | Yes              |
| < 2.8   | No               |

## Reporting a Vulnerability

Email [security@ryanbigg.com](mailto:security@ryanbigg.com) with:

- Description of the vulnerability
- Steps to reproduce
- Attached patch (if available)

Do not open a public GitHub issue for security vulnerabilities.
